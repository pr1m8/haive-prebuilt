Quick Reference
---------------

**Most Popular Agents**::

    # Business intelligence
    from haive.prebuilt.company_researcher import CompanyResearcher
    
    # Content creation  
    from haive.prebuilt.blog_writer_agent import BlogWriterAgent
    
    # Research assistance
    from haive.prebuilt.scientific_paper_agent import ScientificPaperAgent
    
    # Customer service
    from haive.prebuilt.customer_support import CustomerSupportAgent

**Configuration Patterns**::

    from haive.core.engine.aug_llm import AugLLMConfig
    
    # Business/Legal (Precise)
    business_config = AugLLMConfig(temperature=0.1, max_tokens=2000)
    
    # Research/Analysis (Balanced)
    research_config = AugLLMConfig(temperature=0.3, max_tokens=3000)
    
    # Creative (Flexible)
    creative_config = AugLLMConfig(temperature=0.8, max_tokens=4000)

**Deployment Example**::

    # Choose your agents
    agents = [
        CompanyResearcher(name="analyst", engine=business_config),
        BlogWriterAgent(name="writer", engine=creative_config),
        CustomerSupportAgent(name="support", engine=research_config)
    ]
    
    # Deploy individually or in workflows
    for agent in agents:
        result = await agent.arun("Your task here")
